## 🧭 核心推算原则

本 Skill 遵循十条核心纪律，确保推算诚实可靠：

### 1. 实事求是，独立思考
卦象吉凶如实陈述，不粉饰、不讨好、不编吉利话。角色是诚实的推算者，不是心理安慰师。

### 2. 先分析再回答
接到问题后必须：(a) 分析用户真正想问什么 → (b) 选择最适合的方法 → (c) 解释选此方法的原因 → (d) 用户确认后再推算。

### 3. 主动索取必要信息
禁止假设默认值。缺信息就问，不让偷懒导致错误。

### 4. 推到哪说到哪，不准的不硬编
每种方法有精度边界。能精确的就精确，说不准的就范范，卦里没信号的绝不写。

### 5. 一个卦只回答一个问题
「命」和「事」必须严格区分。新问题 = 新卦 = 重新起卦。不同的问题不塞进一个卦里解读。

### 6. 每次排盘后自查
输出「排盘完成，自查中...」→ 对照自检清单 → 确认无误后开始解读。

### 7. 推不了就直接说，给替代方案
六种方法都回答不了时，诚实地说明，并提出 2-3 个能确切回答的替代问题。

### 8. 每个卦象符号给出完整含义谱系
关键卦象/爻象至少给两个方向的解读，不片面、不单选最戏剧化的那一面。

### 9. 解读前确认时间尺度
同一卦在当日/短期/中期/终局尺度上含义不同。解卦前先确认用户问的是哪个尺度。

### 10. 每次新问题先分类：命还是事
「命」用八字/紫微，「事」用小六壬/梅花/六爻/奇门。不能用命盘性格推断具体事件。

## 🚀 快速开始

### 环境要求

- **Python** ≥ 3.10
- **零第三方依赖** —— 全部基于 Python 标准库（`datetime`、`itertools`、`json` 等）
- 无需安装任何包，克隆即用

### 运行测试

```bash
cd scripts
python test_all.py
```

测试覆盖：干支推算、八字排盘、小六壬掌诀、梅花易数起卦、六爻装卦、紫微斗数排盘、奇门遁甲排盘。

### 代码示例

```python
from utils import xiaoliuren, day_ganzhi
from bagua import meihua_from_numbers, meihua_from_datetime, liuyao_auto_qigua
from ganzhi import pai_sizhu, bazi_full

# ─── 小六壬：农历月、日、时掌诀推算 ───
result = xiaoliuren(month=3, day=15, hour=9)
# → {'掌诀': '速喜', '吉凶': '大吉', '属性': '火/朱雀/南方', ...}

# ─── 梅花易数：报三个数字起卦 ───
gua = meihua_from_numbers(upper=5, lower=8, changing=3)
# → 返回本卦、互卦、变卦、体用关系、动爻位置

# 或者用当前时间自动起卦
gua = meihua_from_datetime()

# ─── 八字：排四柱 + 大运 ───
bazi = bazi_full(2000, 1, 1, 12, gender='男')
# → 年柱·月柱·日柱·时柱 + 十神 + 大运起运年龄

# ─── 六爻：自动起卦（或手动传入六次数值） ───
liuyao = liuyao_auto_qigua()
# → 本卦·变卦 + 六亲·六兽·世应·纳支

# ─── 紫微斗数：排盘（需精确到小时） ───
from ziwei import ziwei_pan
pan = ziwei_pan(year=2000, month=1, day=1, hour=12, gender='男')
# → 十二宫 + 百余星曜 + 四化 + 大限

# ─── 奇门遁甲：时家奇门排盘 ───
from qimen import qimen_pan
pan = qimen_pan(year=2025, month=7, day=9, hour=14)
# → 九宫格 + 八门九星八神 + 天盘地盘
```

## 🏛️ 底层知识体系

所有六种方法共用以下基础体系：

### 八卦 & 六十四卦
- **先天八卦**（伏羲）：乾兑离震巽坎艮坤 → 卦数 1-8
- **后天八卦**（文王）：对应八个方位 + 五行
- **万物类象**：天文·地理·人事·身体·动物·器物 → 梅花易数取象核心
- **六十四卦**：八宫卦（乾坎艮震巽离坤兑）→ 每宫八卦（本宫·一世→五世·游魂·归魂）

### 五行系统
- **五行**：木火土金水 → 相生·相克·四时旺衰（旺相休囚死）
- **天干**：甲乙(木)·丙丁(火)·戊己(土)·庚辛(金)·壬癸(水)
- **地支**：寅卯(木)·巳午(火)·辰戌丑未(土)·申酉(金)·亥子(水)
- **天干合化**：甲己合土·乙庚合金·丙辛合水·丁壬合木·戊癸合火
- **地支关系**：六合·三合·六冲·六害·三刑
- **六十甲子**：干支组合循环 → 年·月·日·时的标注体系

### 干支历法
- **年柱**：以立春为界（非公历元旦，非农历正月初一）
- **月柱**：以节气为界（立春→寅月，惊蛰→卯月…）
- **日柱**：六十天一循环
- **时柱**：五鼠遁法 + 时辰地支
- **夏令时处理**：1986-1991 年 4 月中旬至 9 月中旬需减 1 小时

### 术语词典
面向算命新手的完整术语表，每个术语包含「传统含义」和「白话解释」，确保解读结果普通人能看懂。
