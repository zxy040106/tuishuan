# 算命术语词典

本词典为算命小白用户设计，每个术语包含**传统含义**和**白话解释**。在解读中必须使用术语+白话解释的格式。

## 通用术语

| 术语 | 含义 | 白话 |
|------|------|------|
| 干支 | 天干+地支的组合，如"甲子" | 古代中国的计时编码，60个一组循环 |
| 五行 | 木火土金水五种"气"的运行 | 自然界五种基本力量模式 |
| 生克 | 五行相生和相克的关系 | 事物之间促进或制约的关系 |
| 旺衰 | 五行在特定时节的强弱状态 | 在特定时间环境下某个能量的强弱 |
| 用神 | 平衡命局的关键五行/六亲 | 对你命盘最有帮助的那个元素 |
| 忌神 | 破坏命局平衡的五行/六亲 | 对你命盘最有害的那个元素 |
| 格局 | 八字结构的整体分类模式 | 你命盘的基本"类型"标签 |

## 六爻术语

| 术语 | 含义 | 白话 |
|------|------|------|
| 本卦 | 摇卦所得的原卦 | 事情当前面貌 |
| 变卦 | 动爻变化后形成的卦 | 事情发展的结果 |
| 动爻 | 6或9的爻（老阴老阳），会发生变化的爻 | 事情的关键变化点 |
| 静爻 | 7或8的爻（少阳少阴），保持不变的爻 | 事情中稳定的部分 |
| 世爻 | 代表问卦者自身的那一爻 | 卦中的"你" |
| 应爻 | 与世爻隔三位对应的爻（世应间隔三爻），代表对方/客观环境 | 卦中的"对方/所问之事" |
| 用神 | 代表所问事类的爻 | 卦中对应你问题的爻位 |
| 六亲（爻） | 父母/官鬼/子孙/妻财/兄弟（在此指每爻的属性） | 每爻纳支五行与卦宫五行的生克关系 |
| 六兽 | 青龙/朱雀/勾陈/螣蛇/白虎/玄武 | 每爻的六种附加属性，有象征意义 |
| 青龙 | 吉兽，主喜事/贵人 | 好的、喜事的、正面的 |
| 朱雀 | 信息、文书、口才 | 传播的、表达的，也主口舌是非 |
| 勾陈 | 稳定、房产、田土 | 固定的、拖延的、与土地房屋有关 |
| 螣蛇 | 智慧、谋划、变化 | 灵活的、虚惊的、怪异之事 |
| 白虎 | 凶兽，主伤病/血光 | 危险的、伤害的 |
| 玄武 | 谋略、暗中运作 | 潜藏的、暧昧的、欺诈的 |
| 装卦 | 为卦象配置各种属性的过程 | 给卦的每条线标注地支、五行、六亲等 |
| 旬空 | 日柱所在旬空的地支（甲子旬戌亥空等） | 对应地支的爻当前"空"了，需待填实 |
| 月破 | 爻的地支被月建（当月地支）六冲 | 当月力量被冲散，暂时无法发挥作用 |
| 日破 | 爻的地支被日辰六冲且自身衰弱（被日克） | 当前断了、不在位 |
| 暗动 | 爻的地支被日辰六冲但自身旺相（被日生或临日辰） | 被激发反而动起来了 |
| 伏神 | 用神不在本卦六亲中时，到本宫卦查找的对应爻 | 隐藏的力量，需被引出才能发挥作用 |
| 飞神 | 伏神同位之上的原卦爻 | 压在伏神上面的爻，飞生伏则吉 |
| 进神 | 动爻变同五行且地支前进一位（如寅→卯） | 有力前进 |
| 退神 | 动爻变同五行且地支后退一位（如卯→寅） | 衰退无力 |

## 使用要求

**在输出解读时**：
1. 首次出现任何术语，必须附带白话解释，格式为「术语（白话：xxx）」
2. 所有判断必须锚定具体盘面元素，如「本卦官鬼持世（白话：卦中你自身带压力属性），在午火（白话：五行属火），说明...」
3. 不要堆砌术语。一段话中术语不宜超过3个
4. 避免使用没有在本词典中列出的罕见术语。如需使用，要先在解读中给出解释
