## 五行相生

```
木 → 火 → 土 → 金 → 水 → 木
```

| 生 | 被生 | 原理 |
|----|------|------|
| 木 | 火 | 木燃生火 |
| 火 | 土 | 火尽成灰（土） |
| 土 | 金 | 土中产金 |
| 金 | 水 | 金熔成水（或金生水汽） |
| 水 | 木 | 水养树木 |

## 五行相克

```
木 → 土 → 水 → 火 → 金 → 木
```

| 克 | 被克 | 原理 |
|----|------|------|
| 木 | 土 | 树木破土而出 |
| 土 | 水 | 土堤挡水 |
| 水 | 火 | 水能灭火 |
| 火 | 金 | 火能熔金 |
| 金 | 木 | 金属砍伐树木 |

## 五行旺衰（四时）

| 五行 | 春 | 夏 | 秋 | 冬 | 四季末(辰戌丑未月) |
|------|----|----|----|----|---------------------|
| 木 | **旺** | 休 | 死 | 相 | 囚 |
| 火 | 相 | **旺** | 囚 | 死 | 休 |
| 土 | 死 | 相 | 休 | 囚 | **旺** |
| 金 | 囚 | 死 | **旺** | 休 | 相 |
| 水 | 休 | 囚 | 相 | **旺** | 死 |

- **旺**: 当令，力量最强
- **相**: 被生，力量次强
- **休**: 生他，力量减弱
- **囚**: 被克，力量很弱
- **死**: 被旺者克，力量最弱

## 天干合化

| 天干对 | 化气 | 名称 |
|--------|------|------|
| 甲 + 己 | 土 | 中正之合 |
| 乙 + 庚 | 金 | 仁义之合 |
| 丙 + 辛 | 水 | 威制之合 |
| 丁 + 壬 | 木 | 淫匿之合 |
| 戊 + 癸 | 火 | 无情之合 |

> 天干五合有正化与反化之分。命局中如合化成功，以化气论五行；如合而不化，则两支并论。

## 在算命中的应用

- **八字**: 日干五行 + 月支旺衰 + 十神生克 = 格局判断的核心
- **六爻**: 卦宫五行 + 爻支五行 → 六亲确定 + 旺衰判断
- **梅花易数**: 体卦五行 vs. 用卦五行 → 生克吉凶
- **奇门遁甲**: 九星五行 + 八门五行 + 天盘干五行 = 克应判断
