# 节气与历法参考

## 二十四节气

中国农历是阴阳合历，以太阳运行定节气，以月相定朔望。算命所用"月柱"以节气中的"节"为换月点。

> **警告**: 以上为近似日期，每年有±1天的偏差。本Skill的Python脚本使用精确的立春日期表进行判断。

## 年柱边界

年柱以**立春**为界，而非公历1月1日或农历正月初一。

例：2024年2月3日出生，年柱仍为"癸卯"（因立春在2月4日）。
例：2024年2月5日出生，年柱为"甲辰"。

## 时辰换算

| 时辰 | 地支 | 时间范围 |
|------|------|---------|
| 子时 | 子 | 23:00 - 00:59 |
| 丑时 | 丑 | 01:00 - 02:59 |
| 寅时 | 寅 | 03:00 - 04:59 |
| 卯时 | 卯 | 05:00 - 06:59 |
| 辰时 | 辰 | 07:00 - 08:59 |
| 巳时 | 巳 | 09:00 - 10:59 |
| 午时 | 午 | 11:00 - 12:59 |
| 未时 | 未 | 13:00 - 14:59 |
| 申时 | 申 | 15:00 - 16:59 |
| 酉时 | 酉 | 17:00 - 18:59 |
| 戌时 | 戌 | 19:00 - 20:59 |
| 亥时 | 亥 | 21:00 - 22:59 |

> **注意**: 23:00后为次日的子时。如果出生在中国，习惯使用北京时间即可。如果在新疆/西藏等使用地方时的地区，需要说明时区。

## 命理中的时间换算注意事项

1. **真太阳时 vs 北京时间**: 传统命理应以当地真太阳时为准。如用户不确定出生地经度，默认使用北京时间。
2. **子时细分**: 23:00-00:59 = 子时，但23:00-23:59 = 晚子时（起时柱有争议，本Skill统一使用常规的五鼠遁法）。
3. **夏令时**: 提醒用户1986-1991年间中国实行过夏令时，出生在此时段的需减一小时。
