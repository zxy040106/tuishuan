# 奇门遁甲 — 排盘与解盘参考

## 方法概述

奇门遁甲是中国传统术数中最复杂的一种，被称为"帝王之术"。它在一个九宫格中同时排布天（九星）、地（八卦+天干）、人（八门）、神（八神）四层信息，形成一个时空模型。

奇门遁甲的核心价值是**择时择方**——告诉你什么时间、往什么方向、做什么事最有利。

### 适用场景
- "什么时候做这件事最合适？"（择时）
- "往哪个方向发展最好？"（择方）
- "这件事的战略时机判断"
- "当前的困境如何突破？"

### 不适用场景
- 快速判断 → 用小六壬
- 命理分析 → 用八字/紫微
- 不需要时空定位的决策 → 用六爻

## 重要声明

**奇门遁甲流派众多（置闰法、拆补法、茅山法等），对局数的起法各家不同。本Skill采用时家奇门置闰法（最主流的流派）。排盘结果与其他流派可能有差异，特此说明。**

## 所需信息

1. **具体的日期和时间**（精确到小时）—— 奇门以时辰为单位起局
2. **问什么事情** —— 确定判断方向（看哪个宫）

如果用户只是问"现在的情况"，用当前时间起局。

## 九宫

奇门用九宫格，对应后天八卦方位：

```
 4巽 | 9离 | 2坤
-----+-----+-----
 3震 | 5中 | 7兑
-----+-----+-----
 8艮 | 1坎 | 6乾
```

## 八门

八门代表人事的吉凶，是奇门中最重要的判断依据。

| 门 | 五行 | 吉凶 | 含义 | 白话 |
|----|------|------|------|------|
| **休门** | 水 | 吉 | 休息、修养、谒贵、嫁娶 | 宜求见贵人、上任、婚嫁 |
| **生门** | 土 | 大吉 | 生机、财运、发展 | 最旺财运的门 |
| **伤门** | 木 | 凶 | 伤害、竞争、捕盗 | 适合竞争/讨债/竞技；不宜求财 |
| **杜门** | 木 | 中(偏凶) | 堵塞、隐藏、技术 | 宜躲藏/钻研；不宜出行/求人 |
| **景门** | 火 | 中 | 文书、计划、表面 | 适合做方案写文章 |
| **死门** | 土 | 大凶 | 终结、停滞、杀伐 | 不动最好；猎捕可用 |
| **惊门** | 金 | 凶 | 惊恐、口舌、辩论 | 宜官司/捕盗；律师可用 |
| **开门** | 金 | 大吉 | 开创、出行、公开 | 最利事业发展的门 |

> 口诀: **急从伤门走，生门万事成，开门大吉昌，休门万事休，杜门不可往，景门宜献书，死门不可出，惊门宜捕盗。**

## 排盘流程（脚本执行）

_由 `scripts/qimen.py::qimen_pai_pan()` 执行。_

### 核心步骤
1. **定阴遁/阳遁**: 冬至后阳遁，夏至后阴遁
2. **定局数**: 根据日干支所属节气确定是第几局（1-9）
3. **布地盘**: 三奇六仪按局数排列在九宫
4. **定值符星和值使门**: 根据时辰确定
5. **转天盘**: 值符星转到时干所在宫，其他八星顺转
6. **转人盘**: 值使门转到时辰所在宫，其他七门顺转
7. **排八神**: 值符在值符星所在宫，阳遁顺排八神，阴遁逆排

### 自检清单
- [ ] 阴阳遁与节气匹配（冬至到夏至=阳遁，夏至到冬至=阴遁）
- [ ] 局数在1-9之间
- [ ] 地盘三奇六仪布列正确
- [ ] 值符/值使门定位正确
- [ ] 天盘和人盘旋转正确
- [ ] 八神顺逆与阴阳遁匹配

## 解读框架

### 第一步：定方向（用神体系）
根据用户的问题，锁定要看哪个宫：

| 问题类型 | 主要用神 | 兼看 |
|---------|---------|------|
| 事业/求职 | **开门** + 值符 + 日干 | 官鬼星(天心/天芮) |
| 财运/投资 | **生门** + 戊(资金) | 日干 |
| 感情/婚姻 | **六合** + 乙(女)/庚(男) | 日干、时干 |
| 考试/求学 | **天辅星** + 景门 + 丁奇 | 值符、日干 |
| 健康/疾病 | **天芮星**(病) + 时干 | 乙奇(医药)、生门(康复) |
| 出行/远行 | **开门** + 日干 + 时干 | 驿马(寅申巳亥落宫) |
| 寻人/寻物 | **时干** + 杜门 | 日干 |
| 时机选择 | **值符** + 目标宫 | 时干 |
| 自身处境 | **日干落宫** | 时干落宫 |

### 第二步：看宫位四层信息叠合
把目标宫的四层信息叠在一起解读：
天盘干 + 九星 + 八门 + 八神 → 综合判断

### 第三步：看日干与时干的关系
- 日干所在宫 = 你（问卦者）
- 时干所在宫 = 事
- 两宫的生克关系 → 事能否成

### 第四步：看格局
天盘干与地盘干的组合形成"格局"（克应），有些是吉格，有些是凶格。

#### 常见吉格
| 格局 | 组合 | 含义 |
|------|------|------|
| 青龙返首 | 戊+丙 | 事情可成、有大进展 |
| 飞鸟跌穴 | 丙+戊 | 收益好、回报高 |
| 玉女守门 | 值使门落丁奇宫（特定时辰） | 利于暗中运作 |
| 三奇得使 | 乙+己/辛, 丙+戊/庚, 丁+壬/癸 | 吉上加吉，得贵人助 |

#### 全局格局
| 格局 | 条件 | 含义 |
|------|------|------|
| 反吟 | 星门对冲（全盘对冲） | 事情反复、颠倒、速变速败 |
| 伏吟 | 星门原地不动 | 事情停滞、困顿、宜守不宜攻 |
| 五不遇时 | 时干克日干（以时干旺衰定轻重） | 百事皆凶，择时忌用 |

#### 常见凶格
| 格局 | 组合 | 含义 |
|------|------|------|
| 青龙逃走 | 乙+辛 | 宜走不宜留 |
| 白虎猖狂 | 辛+乙 | 凶险突生 |
| 朱雀投江 | 丁+癸 | 文书口舌之祸 |
| 螣蛇夭矫 | 癸+丁 | 事情曲折复杂 |
| 太白入荧 | 庚+丙 | 贼寇必来（敌动） |
| 荧入太白 | 丙+庚 | 贼寇必退（敌退） |
| 伏宫格 | 庚+值符 | 处处受阻 |
| 飞宫格 | 值符+庚 | 步步艰难 |
| 六仪击刑 | 地盘六仪落宫自刑（如戊落震宫=子卯刑） | 事情反复折腾、伤人 |
| 大格 | 庚+癸 | 百事不利，求人难成 |
| 小格 | 庚+壬 | 出行不利，主客皆伤 |
| 刑格 | 庚+己 | 主客两伤，刑狱词讼 |

### 第五步：应期
- 看目标宫的地支 → 那个支的月/日就是应期
- 看日干宫与目标宫的距离（隔几位 → 几天/几月/几年）

## 五行生克在奇门中的特殊应用

奇门不只看生克本身，还看**旺衰**和**门星配合**：
- 吉门克凶星 → 可用（正压邪）
- 凶门生吉星 → 虽凶可缓
- 吉门生凶星 → 好事帮坏事
- 凶门+凶星落旺宫 → 加倍凶险
