# 八字（四柱命理）— 排盘与解盘参考

## 方法概述

八字是分析出生时间（年月日时）对应天干地支组合的命理学方法。通过分析八个字的五行生克、十神关系、格局高低、大运走势，推断一个人的性格、运势和人生走向。

八字不是"算命"而更像是"读命"——从出生时间的干支配置中读出先天倾向和后天走势。

### 适用场景
- 了解自己的性格特点和天赋
- 分析一生运势起伏（事业、财运、感情、健康）
- 大运流年的具体走势判断
- 择业方向、适合发展的领域

### 不适用场景
- "明天能不能中彩票"（具体事件的应期判断用六爻更合适）
- 快速吉凶判断（用小六壬）
- 与生辰无关的具体决策（用六爻/梅花）

## 所需信息（向用户索取）

### 必须信息
1. **出生年份**（公历）
2. **出生月日**（公历）
3. **出生时间**（尽可能精确到小时，24小时制）
4. **性别**（男/女——大运顺逆取决于此）

### 提醒事项
- 如果只记得"早上""下午"等模糊时间 → 询问是否能缩小范围。实在不确定则标注影响
- 1986-1991年间中国实行夏令时 → 提醒用户出生在4月中旬至9月中旬的需减1小时
- 时区问题 → 默认北京时间，如果用户在新疆/西藏等使用地方时的地区需说明

### 可以缺省的情况
- 没有出生时辰 → 仍可排年/月/日三柱（六字），但无时柱则无法断子女和晚年运
- 性别未知 → 大运顺逆无法确定，严重影响解读，建议追问

## 排盘流程（脚本执行）

_由 `scripts/ganzhi.py::bazi_full()` 执行。_

### 步骤1: 排四柱
见 `references/shared/calendar.md` 和 `references/shared/ganzhi.md`

核心要点：
- 年柱以立春为界（非公历元旦，非农历正月初一）
- 月柱以节为界（立春→寅月，惊蛰→卯月...）
- 日柱按60天一循环（脚本基于1900-01-01=甲戌计算）
- 时柱按五鼠遁 + 时辰地支确定

### 步骤2: 排十神
以日干（日主）为"我"，看其他七个天干与我的关系。

| 关系 | 同阴阳 | 异阴阳 |
|------|--------|--------|
| 同五行 | 比肩 | 劫财 |
| 生我 | 偏印 | 正印 |
| 我生 | 食神 | 伤官 |
| 克我 | 七杀 | 正官 |
| 我克 | 偏财 | 正财 |

### 步骤3: 排大运
- 阳年男 + 阴年女 → 顺排（月柱之后顺推）
- 阴年男 + 阳年女 → 逆排（月柱之前逆推）
- 起运岁数 = 出生到下一个（或上一个）节的天数 ÷ 3

### 步骤4: 查神煞
见 `scripts/ganzhi.py::find_shensha()`

## 自检清单

排盘完成后必须检查：
- [ ] 年柱立春边界正确（2月3-5日出生需特别注意）
- [ ] 月柱节气边界正确（月头月尾出生需特别注意）
- [ ] 日柱可通过已知参考日期交叉验证
- [ ] 时柱子时（23:00-00:59）归属正确
- [ ] 大运顺逆与年干阴阳+性别匹配
- [ ] 十神关系以日干为基准
- [ ] 纳音五行全部在位

**如果日柱计算不自信，用另一个参考日期重算一遍。**

## 解读框架

### 第一层：日主定性
- 日干是什么五行 → 基本性格底色
- 日支是什么 → 内心世界和配偶宫
- **日主的旺衰**（四维判断）→ 精力强弱

#### 旺衰判断（四维标准）
日主旺衰不能仅看月令，需综合四个维度：
- **得令**: 日主五行在月令是否旺相（寅卯月木旺、巳午月火旺、申酉月金旺、亥子月水旺、辰戌丑未月土旺）
- **得地**: 地支中是否有日主的根气（比劫）或印星生扶。日支、月支、年支均需考察
- **得生**: 天干是否有印星（正印/偏印）透出生扶日主
- **得助**: 天干是否有比肩/劫财透出帮扶日主

判断标准：
- 四得俱全 → 身极旺，要克泄耗
- 仅得令但不得地不得生 → 看似旺实弱，需扶
- 仅得地但月令失令 → 有根气不弱
- 一得都没有 → 身弱，从弱还是扶弱需看地支有无微根

> ⚠️ **重要**: 极弱无根无生 → 从弱格，不能硬扶；极旺不可克 → 犯旺反凶

### 第二层：十神格局
- 月支本气是什么十神 → 格局类型（正格）
  - 正官格 → 规矩守法、适合体制
  - 七杀格 → 果断魄力、适合竞争
  - 正印格 → 温厚仁慈、适合学术
  - 偏印格 → 独特思维、适合技术
  - 食神格 → 才艺温和、适合创意
  - 伤官格 → 聪明锐利、适合表达
  - 正财格 → 勤劳务实、适合经营
  - 偏财格 → 慷慨大方、适合投资
  - 建禄格 → 自立自强（月支为日主禄位）
  - 阳刃格 → 刚强激烈（月支为日主羊刃）
- **格局判定优先级**: 月支本气 → 若本气不透则看月支藏干何者在天干透出 → 取透出者为格
- **特殊格局**（当命局偏枯至极时需要识别）:
  - **从格**: 日主极弱无根无生 → 从财/从杀/从儿/从势，不能扶只能顺势
  - **化气格**: 日主与月干/时干五合，且地支化神旺 → 按化气后的五行论
  - **专旺格**: 命局一派同类五行 → 曲直格(木)、炎上格(火)、稼穑格(土)、从革格(金)、润下格(水)
  - 特殊格局的判断需谨慎，建议结合全局综合判断后再定性

### 第三层：用神判断
找到平衡命局的关键五行：
- **扶抑法**: 
  - 日主弱但有根 → 用生扶日主的五行（印星/比劫）
  - 日主强但可克 → 用克泄耗日主的五行（官杀/食伤/财星）
  - ⚠️ 日主极弱无根（从格）→ 不能扶，要顺势用旺神
  - ⚠️ 日主极旺不可克（专旺）→ 不能克，用顺其势的五行
- **调候法**: 按出生月份取用（参考《穷通宝鉴》）:
  - 寒冬（亥子丑月）→ 一般用火调候，但需看具体日干和月份的配合
  - 炎夏（巳午未月）→ 一般用水调候，但需看具体日干和月份的配合
  - 春秋 → 金水伤官喜见官（申月）、木火伤官喜见印等，不能一概而论
- **病药法**: 
  - 命局中某五行过旺为"病" → 克它或泄它的五行为"药"=用神
  - 某五行被严重克伤为"病" → 生扶该五行的为"药"
  - 某五行缺失也为"病" → 大运流年遇到时起作用
- **通关**: 两五行相战时（如金木交战、水火相冲），引入中间五行化解。如金木相战用水通关（金生水→水生木）
- **喜神与忌神**: 找到用神后，生用神的为喜神，克用神的为忌神。用忌一体两面，缺一不可

### 第四层：大运走势
- 当前处于哪步大运
- 大运天干地支各是什么十神 → 这十年的主题
  - 大运天干管前5年（显现的、外在的变化）
  - 大运地支管后5年（实质的、内在的变化）
- **大运与命局的互动**:
  - 大运干支与命局相同（伏吟）→ 重大变化/重复
  - 大运干支与命局相冲（反吟）→ 动荡/转折
  - 大运生扶用神 → 好运十年
  - 大运生扶忌神 → 逆境十年
- **岁运并临**: 流年干支与大运干支完全相同 → 此年吉凶加倍，需特别关注

### 第五层：流年判断
- 当前流年干支
- **流年→大运→命局 的互动顺序**: 流年先与大运作用，再共同作用于原命局
- 流年天干/地支与日主和用神的关系
- 流年与命局的冲合:
  - 流年冲日支（夫妻宫）→ 感情/婚姻有变动
  - 流年合用神 → 好事来临
  - 流年冲用神 → 好事被冲散
  - 流年与命局三合/三会 → 形成新的力量格局

## 十神在四柱不同位置的含义

同一个十神，出现在年/月/日/时不同柱位，含义完全不同：

| 十神 | 年柱 | 月柱 | 日柱（配偶宫） | 时柱 |
|------|------|------|-------------|------|
| 正官 | 祖上有官职 | 青年得志/有管理权 | 配偶端正 | 晚年有威望/子女成才 |
| 七杀 | 祖上有军警/武职 | 青年压力大/拼搏 | 配偶强势 | 晚年仍有压力/子女严苛 |
| 正印 | 出身书香 | 学业顺利/长辈荫庇 | 配偶善良体贴 | 晚年安稳 |
| 偏印 | 祖上有特殊技能 | 学业偏门/独特思维 | 配偶个性独特 | 晚年孤独/特殊爱好 |
| 食神 | 幼年有福 | 青年才华展现 | 配偶温和 | 晚年安逸/子女孝顺 |
| 伤官 | 幼年叛逆 | 青年锋芒毕露 | 配偶有才但可能挑剔 | 子女聪明有个性 |
| 正财 | 祖上经商 | 青年能赚钱 | 配偶贤惠（男命） | 晚年富足 |
| 偏财 | 祖上有意外财 | 青年横财运 | 人缘好（男命偏缘） | 晚年慷慨/意外财 |
| 比肩 | 兄弟姐妹多 | 青年自强 | 配偶独立 | 晚年仍要奋斗 |
| 劫财 | 幼年有竞争者 | 青年被分财分机会 | 配偶可能被争夺 | 晚年需防破财 |

> ⚠️ 此表是倾向而非绝对定论。具体判断需结合该柱干支是否是用神/忌神、该十神旺衰、以及是否被冲合。

### 正官（克我，异阴阳）
- **正面**: 正直、守规矩、责任感、管理能力
- **负面**: 保守、压力大、受约束
- **代表**: 上司、政府、规则、女性正缘（夫星）

### 七杀（克我，同阴阳）
- **正面**: 魄力、决断力、竞争力、行动力
- **负面**: 暴躁、压力、挑战、危险
- **代表**: 对手、困难、强势上司、女性偏缘

> **食神制杀**: 七杀被食神克制 = 有能力的人能驾驭压力，反而成材

### 正印（生我，异阴阳）
- **正面**: 仁慈、学识、包容、贵人
- **负面**: 懒惰、依赖、缺乏主见
- **代表**: 母亲、老师、学历、庇护

### 偏印（生我，同阴阳）
- **正面**: 独特才能、领悟力、技术天赋
- **负面**: 孤僻、古怪、不合群
- **代表**: 继母、特殊技能、偏门学问

### 食神（我生，同阴阳）
- **正面**: 才华、温和、创造力、口福
- **负面**: 散漫、过于理想化
- **代表**: 子女（女命）、才艺、享受

### 伤官（我生，异阴阳）
- **正面**: 聪明、锐利、表达能力、艺术天赋
- **负面**: 叛逆、刻薄、不服管
- **代表**: 才华流露、创新突破

> **伤官见官**: 伤官与正官同现 = 不服管束，容易与人冲突。传统认为不利官运

### 正财（我克，异阴阳）
- **正面**: 勤俭、踏实、正当收入
- **负面**: 计较、小气、过于务实
- **代表**: 薪水收入、男性正缘（妻星）

### 偏财（我克，同阴阳）
- **正面**: 慷慨、大方、意外之财、人缘
- **负面**: 挥霍、不稳、好投机
- **代表**: 横财、投资收入、父亲、男性偏缘

### 比肩（同我，同阴阳）
- **正面**: 自立、有主见、自尊
- **负面**: 固执、自我中心
- **代表**: 兄弟姐妹、同辈、自我

### 劫财（同我，异阴阳）
- **正面**: 社交能力强、行动力
- **负面**: 竞争、破财、冲动
- **代表**: 竞争者、朋友可能变对手

## 地支藏干的解读意义

地支藏干揭示了表面之下的深层信息。比如日支（配偶宫）的藏干揭示了配偶的真实性格。

> 例: 日支为"寅"，藏甲（本气）+丙+戊。
> - 表面地支寅=木=温和，但藏干有丙（火）=内心热情，戊（土）=实际
> - 合在一起解读：表面上随和好相处，内心其实很有主见和行动力。

## 合冲刑害的解读

| 关系 | 正面 | 负面 |
|------|------|------|
| 天干五合 | 化解矛盾、有缘、合作 | 失去自我、被牵制 |
| 地支六合 | 缘分深、亲密 | 被束缚 |
| 地支六冲 | 动力、变动（坏事冲走） | 不稳定、离散 |
| 地支三合 | 团结力、大缘分 | - |
| 地支三刑 | - | 纠结、伤害、诉讼 |
| 地支六害 | - | 暗中破坏、误解 |

## 八字对感情的解读

当用户询问感情/婚姻时，重点看：

### 男命看妻星
- **正财** = 正妻，**偏财** = 偏缘/女友
- 财星旺 → 异性缘好
- 财星弱或被克 → 姻缘来得晚或需努力
- 财星多现 → 异性缘复杂
- 日支（配偶宫）看配偶本人特征

### 女命看夫星
- **正官** = 正夫，**七杀** = 偏缘
- 同理判断旺衰和数量

### 共通看配偶宫
- 日支的五行+十神解读配偶的性格
- 日支被冲 → 婚姻有波动
- 日支被合 → 配偶与人有牵连（或夫妻关系亲密）

## 性格判断法

从日干五行定基本性格底色：

| 日干五行 | 基本性格 |
|---------|---------|
| 甲木 | 正直向上、有领导力、如大树般可靠 |
| 乙木 | 柔韧适应、善交际、如藤蔓般灵活 |
| 丙火 | 热情奔放、光明磊落、如太阳般温暖 |
| 丁火 | 细腻持久、内心温暖、如烛火般内敛 |
| 戊土 | 稳重厚道、诚信可靠、如大地般踏实 |
| 己土 | 细腻包容、善于照顾、如田园般滋养 |
| 庚金 | 刚毅果断、有魄力、如刀剑般锋利 |
| 辛金 | 精致讲究、追求完美、如珠宝般贵气 |
| 壬水 | 豁达大度、善于变通、如江河般开阔 |
| 癸水 | 细腻敏感、直觉强、如雨露般温柔 |

再从十神组合中细化：
- 正官多 → 守规矩、重责任
- 七杀多 → 竞争心强、不服输
- 食神多 → 温和爱吃、有才艺
- 伤官多 → 聪明锐利、有个性
