# 小六壬 — 掌诀推算参考

## 方法概述

小六壬是最简单的中国传统占卜法，通过三个时间参数（月、日、时）在左手掌六个位置间推算。适合快速询问吉凶、时机等简单问题。

### 适用场景
- 问事情能否成功（是否、吉凶判断）
- 问行人何时归来
- 问失物能否找回
- 问短期内的事情走向

### 不适用场景
- 需要详细分析的人生规划
- 复杂的决策（多个选项比较）
- 需要具体行动建议的问题

## 所需信息（必须向用户确认）

1. **农历月**（1-12）：如用户只知公历日期，用脚本转换
2. **农历日**（1-30）：同上
3. **时辰**（可选，用于更精确的判断）：子/丑/寅/卯/辰/巳/午/未/申/酉/戌/亥

> 如果用户只知道公历日期，运行 `scripts/ganzhi.py` 中的 `determine_nongli_month()` 进行转换，或直接运行 `from utils import xiaoliuren` 函数。

## 六掌诀详解

### 掌诀位置（顺数方向）
大安(1) → 留连(2) → 速喜(3) → 赤口(4) → 小吉(5) → 空亡(6) → 回到大安

### 各掌诀解读表

#### 大安（属木，青龙，东方）
- **含义**: 平安稳定，诸事顺遂
- **吉凶**: 大吉
- **问事**: 事可成，但宜守不宜攻
- **问财**: 平稳得财，不宜贪多
- **问行人**: 行人平安，按计划归来
- **问失物**: 未丢远，可找回
- **应期**: 1/5/7日内
- **性格特征**: 沉稳、保守、可靠

#### 留连（属水，玄武，北方）
- **含义**: 纠缠拖延，事难速决
- **吉凶**: 凶
- **问事**: 事有阻碍，需耐心等待
- **问财**: 财运阻滞，不宜投资
- **问行人**: 行人拖延，有羁绊
- **问失物**: 物品被掩盖，难找
- **应期**: 2/8/10日内
- **性格特征**: 黏滞、犹豫、放不下

#### 速喜（属火，朱雀，南方）
- **含义**: 喜庆迅速，求谋有成
- **吉凶**: 大吉
- **问事**: 事可速成，时机正好
- **问财**: 财运快速到来
- **问行人**: 能迅速回来/到达
- **问失物**: 快快寻找，可见到
- **应期**: 3/6/9日内
- **性格特征**: 急躁、活泼、行动力强

#### 赤口（属金，白虎，西方）
- **含义**: 口舌是非，破财官讼
- **吉凶**: 凶
- **问事**: 有口舌争执，小人作梗
- **问财**: 有破财风险，慎言慎行
- **问行人**: 途中不顺，恐有纠纷
- **问失物**: 被他人所得，难追回
- **应期**: 4/7/10日内
- **性格特征**: 言辞激烈、易冲突

#### 小吉（属水，六合，北方）
- **含义**: 行人喜至，凡事和合
- **吉凶**: 吉
- **问事**: 合作顺利，能成
- **问财**: 合作生财，和气生财
- **问行人**: 行人将至，有喜讯
- **问失物**: 可向北方寻找
- **应期**: 1/4/7日内
- **性格特征**: 和气、合作、人缘好

#### 空亡（属土，勾陈，中）
- **含义**: 谋事落空，劳而无成
- **吉凶**: 大凶
- **问事**: 事不可为，宜暂缓
- **问财**: 求财不成，反有损失
- **问行人**: 音信全无
- **问失物**: 已丢失，难寻回
- **应期**: 3/6/9日内
- **性格特征**: 空想、不落地、虚无

## 推算步骤

### 算法（脚本实现，见 `scripts/utils.py::xiaoliuren()`）

1. **月上起日**: 正月从大安（寅位）数起，顺数至出生月
   - 正月 = 大安 (0), 二月 = 留连 (1), ..., 六月 = 空亡 (5), 七月 = 大安 (0), ...
   - 公式: `month_pos = (lunar_month - 1) % 6`

2. **日上起时**: 从月落位起初一，顺数至出生日
   - 从 month_pos 位为"第1天"，顺数至第 lunar_day 天
   - 公式: `day_pos = (month_pos + lunar_day - 1) % 6`

3. **时上定局**: 从日落位起子时，顺数至出生时辰（可选）
   - 从 day_pos 位为"子时"，顺数至实际时辰

### 自检清单

脚本执行后，检查以下内容：
- [ ] 输入的月、日、时是否在合理范围（月1-12, 日1-30）
- [ ] 三次推算是否都落在六个掌诀之一
- [ ] 最终落位的掌诀属性是否与五行/方向匹配
- [ ] 如果用户提供的是公历日期，是否已正确转换为农历

## 解读原则

1. **直接明了**: 小六壬不是复杂系统，不要过度解读。六个掌诀各有所主，结果简明。
2. **结合问题**: 同一个掌诀对于不同问题有不同侧重。如"留连"问感情是放不下，问出行则是拖延。
3. **中间过程也有用**: 月上起日和日上起时中间的掌诀也蕴含信息，可以提及但不喧宾夺主。
4. **吉凶明确**: 小六壬不模棱两可。大吉就是大吉，大凶就是大凶——不要为了"不让用户难过"而粉饰空亡的结果。

```
## 小六壬 推算结果

### 一、推算过程
- 农历X月 → 月上起日落位: [掌诀]
- 农历X日 → 日上起时落位: [掌诀]
- X时 → 时上定局最终落位: **[掌诀]**

### 二、最终结果：[掌诀]（吉/凶）
解释掌诀含义（白话+针对问题的具体说明）

### 三、行动建议
基于掌诀的建议

### 四、注意事项
- 本次推算的局限性：小六壬为简易占法，精度有限。仅适用于快速吉凶判断，不适合深度分析。
- 信息不确定的影响：如用户只提供了公历日期，农历转换存在±1天误差，可能影响落位。
- 算命是传统文化参考，不是科学预测，不可代替理性决策。
```
