# 六爻（纳甲筮法）— 装卦与解卦参考

## 方法概述

六爻是最系统、最细致的占卜方法。通过六次摇卦（或自动起卦）得一卦，经过"装卦"（装配各种属性），找到"用神"，分析生克旺衰来判断事情的吉凶成败。

### 适用场景（最优选择）
- "我应该做这件事吗？"（决策型问题）
- "A和B选哪个？"（二选一）
- "这件事的走势如何？"（趋势判断）
- "某某人/事对我有利吗？"（关系判断）

### 不适用场景
- 需要快速简单答案 → 用 小六壬
- 从外部观察/灵感推测 → 用 梅花易数
- 一生的命理运势 → 用 八字/紫微

## 所需信息（向用户索取）

### 1. 摇卦方式
- **推荐手摇**：引导用户用三枚硬币摇六次
- **备选自动**：用户不方便时自动起卦（需清晰标注）

### 2. 手摇引导语

> "请准备三枚一样的硬币。心中默念你要问的事情，越具体越好。
>
> 每摇一次：有字的一面 = 正面 = **3**，有花的一面 = 反面 = **2**。
> 三枚加在一起就是一个数字（6/7/8/9）。
>
> 共摇**六次**，第一次是初爻，第六次是上爻。
> 请按顺序报给我六个数字。例如：7, 8, 6, 7, 7, 9"

### 3. 如果用户自己摇
让用户逐次报数字，每次确认后再继续。不要催。

### 4. 占卦日期
使用当日的干支（脚本自动获取）。占卦日期影响六兽排布和旺衰判断。

## 装卦流程（脚本执行）

_由 `scripts/bagua.py::liuyao_construct()` 和 `liuyao_zhuanggua()` 执行。_

### 步骤1: 构建本卦和变卦
- 六次摇卦的数值（6=老阴, 7=少阳, 8=少阴, 9=老阳）
- 6和9为动爻，7和8为静爻
- 本卦：阳(7,9)阴(6,8) → 上下卦 → 六十四卦之一
- 变卦：动爻变化（6阴→阳, 9阳→阴）→ 上下卦 → 六十四卦之一

### 步骤2: 寻宫（确定卦所属的八宫）
- 查六十四卦表，确定卦属哪个宫（乾/坎/艮/震/巽/离/坤/兑）
- 卦宫五行为"我"，决定所有爻的六亲关系

### 步骤3: 安世应
- 查表确定世爻位置（1-6）
- 应爻 = 世爻 + 3（超过6则减6）

### 步骤4: 纳支（每爻配上地支）
- 下卦纳支取对应八卦的纳支表前三条
- 上卦纳支取对应八卦的纳支表后三条
- 阳卦纳阳支，阴卦纳阴支

### 步骤5: 纳六亲
- 对每爻的纳支五行与卦宫五行比较：
  - 爻生宫 = 父母（爻支五行生卦宫五行 → 爻是宫的"父母"）
  - 宫生爻 = 子孙（卦宫五行生爻支五行 → 爻是宫的"子孙"）
  - 爻克宫 = 官鬼
  - 宫克爻 = 妻财
  - 同五行 = 兄弟

### 步骤6: 安六兽
- 按占卦日的日干定初爻六兽
- 甲/乙→青龙, 丙/丁→朱雀, 戊→勾陈, 己→螣蛇, 庚/辛→白虎, 壬/癸→玄武
- 从初爻顺排至上爻

## 自检清单（装卦后必须逐条检查）

装卦完成后，必须逐条验证：

- [ ] 本卦和变卦都在六十四卦中
- [ ] 卦宫正确（查六十四卦表）
- [ ] 世爻位置正确（查表验证）
- [ ] 应爻位置 = （世爻 + 3）% 6（取1-6）
- [ ] 六爻纳支序列正确（阳卦纳阳支，阴卦纳阴支）
- [ ] 六亲关系正确（以卦宫五行为基准）
- [ ] 六兽起始正确（以日干定）
- [ ] 动爻标记与原始摇卦记录一致

**如果任何一条不通过，停止解卦，先定位并修复问题。**

## 解卦逻辑

### 第一步: 定用神

用神是卦中代表所问之事的爻。根据用户的问题类型选择：

| 问题类型 | 用神 | 白话 |
|---------|------|------|
| 事业/工作/官司 | **官鬼爻** | 代表上司、官方、规则 |
| 财运/财产/物资 | **妻财爻** | 代表财富、收入 |
| 父母/文书/学业/考试 | **父母爻** | 代表长辈、文件、考试成绩 |
| 子女/创作/娱乐 | **子孙爻** | 代表晚辈、作品、乐趣 |
| 合伙/竞争/朋友 | **兄弟爻** | 代表同辈、合伙人、竞争者 |
| 自身状态 | **世爻** | 代表你自身 |
| **感情/婚姻（男问）** | **妻财爻** + 应爻 | 妻财为男命之妻，应爻为对方 |
| **感情/婚姻（女问）** | **官鬼爻** + 应爻 | 官鬼为女命之夫，应爻为对方 |
| **健康/疾病** | **官鬼爻**(病) + **子孙爻**(医药) | 官鬼为病症，子孙为康复力 |
| **寻人/寻物** | **父母爻**(信息) + **妻财爻**(失物) | 视具体情况选 |
| **求名/考试** | **官鬼爻**(功名) + **父母爻**(成绩) | 官父双看 |
| **出行/旅行** | **世爻**(自身) + **应爻**(目的地) | 还需看驿马 |

> **伏神**: 当用神不在本卦六亲中任一爻出现时（用神不上卦），需到本宫卦中寻找对应六亲所在的爻——该爻即为伏神，藏在同位爻下方。伏神不得位时力量大减，需待大运/流年引出才能发挥作用。

### 第二步: 看用神状态（旺衰判断）

判断用神爻的旺衰：
1. **月令**: 用神爻的纳支在当月月支的状态（同月支=旺，被月支生=相，被月支克=死，生月支=休，克月支=囚）
   - **月建司令**: 月支是当月的"司令"，对全局有绝对影响力。爻被月建冲 → **月破**（当月力量被冲散），爻被月建合 → **月合**（被锁住）
2. **日辰**: 用神爻的纳支与当日日支的关系（同理）
   - 日冲+旺→**暗动**（被激发），日冲+衰→**日破**（断掉、不在位）
3. **动爻作用**: 是否有其他动爻生/克用神

**旺衰排序**: 旺 > 相 > 休 > 囚 > 死

### 第三步: 看旬空

- 日柱所在旬空哪两个地支（装卦结果中 `旬空` 字段已标注）
- 用神或世爻纳支是否旬空 → 若是，则该爻当前"形同虚设"，需待出空（填空或冲空）方能发挥作用
- 旬空之爻被日冲 → 冲空则实（旬空状态下被冲反而坐实）
- **出空规则**: 旬空之地支到来的日/月（填空），或与旬空之地支六冲的日/月（冲空）

### 第四步: 看世应+用神关系

世爻=你，应爻=对方/环境，用神=所问之事。三者的关系揭示大局。

**世爻与用神的关系**：

| 关系 | 解读 |
|------|------|
| 世爻就是用神 | 事情由你掌控，自身即为事主 |
| 用神生世爻 | 事情对你有帮助，自然得利 |
| 用神克世爻 | 事情对你不利，有压力 |
| 世爻生用神 | 你在付出，需要主动投入 |
| 世爻克用神 | 你可以掌控此事，但需努力 |
| 世爻与用神同五行 | 人与事和谐一致 |

**世爻与应爻的关系**（问合作/感情/竞争时尤其重要）：

| 世应关系 | 含义 |
|----------|------|
| 世生应 | 你主动付出，对方受益 |
| 应生世 | 对方主动对你好/环境有利 |
| 世克应 | 你掌控对方/局面 |
| 应克世 | 对方/环境对你不利 |
| 世应比和 | 双方和谐/立场一致 |

### 第五步: 看动爻

- **动爻本身**: 动爻所在的六亲揭示变化的动力来源
  - 父母爻动 → 文书/长辈因素是变数
  - 官鬼爻动 → 工作/压力因素在变化
  - 妻财爻动 → 金钱/感情因素在变化
  - 子孙爻动 → 创意/下属因素在变化
  - 兄弟爻动 → 竞争/合伙因素在变化

- **动爻对用神的作用**: 动爻生用神=吉，动爻克用神=凶

- **动变关系**（动爻自身的变化方向，装卦结果中 `动变关系` 字段已标注）:
  - **回头克**: 变爻五行克动爻 → 大凶，动爻代表的人/事自身受损
  - **回头生**: 变爻五行生动爻 → 大吉，动爻自身力量增强
  - **化进神**: 动爻变同五行且地支前进（如寅→卯）→ 有力前进
  - **化退神**: 动爻变同五行且地支后退（如卯→寅）→ 衰退无力

- **伏神**: 用神不上卦时，到本宫卦查找对应六亲（装卦结果中 `伏神` 字段已自动计算）:
  - 飞来生伏（飞神生伏神）→ 吉，伏神得长生
  - 飞来克伏（飞神克伏神）→ 凶，伏神被压制
  - 伏来克飞（伏神克飞神）→ 出暴，透出时反而有力
  - 伏来生飞（伏神生飞神）→ 泄气，需待时机

### 第六步: 看六兽加持

| 六兽 | 属性 | 正面含义 | 负面含义 |
|------|------|---------|---------|
| 青龙 | 木 | 喜事、贵人、升迁 | - |
| 朱雀 | 火 | 文书、口才、信息 | 口舌是非 |
| 勾陈 | 土 | 稳定、房产、田土 | 拖延、固执 |
| 螣蛇 | 火 | 智慧、谋划、变化 | 虚惊、怪异 |
| 白虎 | 金 | 刚强、果断、权威 | 伤病、血光 |
| 玄武 | 水 | 谋略、暗中运作 | 欺骗、暧昧 |

**注意**: 六兽是附加信息，不可喧宾夺主。先看用神旺衰和生克，再以六兽做润色。

### 第七步: 看冲合

- **月破**: 用神被月支冲 → 当月力量被冲散，最凶的信号之一
- **日破**: 用神被日支冲且自身衰弱 → 当前断了、不在位
- **暗动**: 用神被日支冲但自身旺（得日生或临日辰）→ 反而被激发，暗中有动作
- **月合/日合**: 用神与月支/日支相合 → 事情被合住，或得外力帮助
- **动爻与用神冲**: 事情有变故
- **动爻与用神合**: 事情被调和
- **三合局**: 三个爻的地支形成三合局（申子辰/亥卯未/寅午戌/巳酉丑）→ 形成了新的力量格局，力量远大于单爻
  - 如用神在三合局中 → 事情有强大合力推动
  - 如忌神在三合局中 → 阻力巨大
- **六合**: 卦中两个地支六合 → 关系紧密、不易分开

> **注意区分月破和日破**: 月破是月建冲（当月破败），日破是日辰冲+衰（当场断裂）。日辰冲+旺=暗动不是日破。月破比日破更严重，因为月建管整月。

### 第八步: 看变卦走向

- 本卦 → 变卦的转变方向：
  - 变卦的卦名和含义暗示事情最终走向
  - 动爻变化后的六亲变化揭示本质改变
- **独发**: 六爻中仅一爻动 → 此爻是事情最关键的变数，全身心聚焦此爻
- **独静**: 六爻中仅一爻静 → 此爻代表的方面最为安稳，不受波及

## 应期判断（事情何时发生）

应期判断较复杂，以下是基本原则：

| 规则 | 说明 |
|------|------|
| 用神旺相且静 | 待用神受生之时（用神被生扶的月/日） |
| 用神休囚且静 | 待用神当令之时（用神五行当旺的季节） |
| 用神为动爻 | 待动爻被合之时（地支合住动爻之日） |
| 用神旬空 | 待出空之时（**填空**: 旬空之地支到来的日/月；**冲空**: 与旬空之地支六冲的日/月） |
| 用神被合 | 待冲开之时（与合住之地支六冲的日/月） |

> **旬空出空规则**: 例：甲子日占卦 → 甲子旬戌亥空。如用神纳支为戌，则待戌日（填空）或辰日（冲戌）应事。

> **重要**: 应期判断是六爻中最难的部分。如果不能确定，如实告诉用户"应期不易判断"，不要硬编一个日子。

## 解读原则

1. **用神为本**: 一切围绕用神展开。用神旺=可为，用神衰=难成，用神被克=有灾
2. **保持诚实**: 卦象显示凶就是凶。不要因为"怕用户难过"而隐瞒或美化
3. **锚定事实**: 每条判断必须指出依据——"因为用神在午火，当前午月为旺，所以..."——而不是空说吉凶
4. **克制发散**: 六爻信息丰富，但不要每个爻都展开解读。聚焦用神+世爻+动爻
5. **动爻优先**: 有动爻的爻位信息量最大，优先解读
