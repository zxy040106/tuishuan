# 紫微斗数 — 排盘与解盘参考

## 方法概述

紫微斗数是中国传统命理学中最精密、信息量最大的系统之一。通过安放一百多颗星曜到十二个宫中，每宫代表人生的一个领域，星曜的性质+所在宫位+四化状态综合判断。

### 适用场景
- 深度个人分析（性格、天赋、优缺点）
- 十二个人生领域的全方位解读
- 人际关系（夫妻、子女、兄弟、朋友）
- 事业方向和财富格局
- 人生各阶段的重点（大限轮转）

### 不适用场景
- 快速判断单一事件 → 用六爻或小六壬
- 需要简便入门的命理 → 用八字

## 所需信息

与八字完全相同，且**出生时间必须精确到小时**。时辰差一个（2小时），命宫就错一个，整个盘就全错了。

特别强调：时辰不确定 → 紫微斗数无法精确推算。必须告知用户这一点。

## 四化

由出生年天干（年干）决定某些主星被"四化"——改变它的作用方向。

| 四化 | 含义 | 白话 |
|------|------|------|
| 化禄 | 增加、得到、顺利 | 好处变多 |
| 化权 | 掌控、权力、主导 | 你能掌控 |
| 化科 | 名声、文采、贵人 | 获得认可 |
| 化忌 | 执着、不顺、缺失 | 容易卡住（但不一定是坏事——忌也是动力） |

> **重要**: 化忌不是"坏"，而是"执"——被化忌的星在的那个宫，是你这辈子最在意、最容易焦虑但也最用心的地方。

## 排盘（脚本执行）

_由 `scripts/ziwei.py::ziwei_pailiang()` 执行。_

### 安星诀核心步骤
1. 确定命宫位置: 从寅宫起正月，顺数至出生月→逆数至出生时
2. 确定五行局: 命宫干支决定（水二局/木三局/金四局/土五局/火六局）
3. 安紫微星: 根据五行局和农历生日确定紫微星位置
4. 安紫微系六星: 紫微定位后按固定顺序逆行安天机等五星
5. 安天府星: 紫微与天府对称分布在寅申线上
6. 安天府系七星: 天府定位后按固定顺序顺行安六星
7. 安辅星: 文昌/文曲/左辅/右弼/天魁/天钺等
8. 安煞星: 擎羊/陀罗/火星/铃星等
9. 定四化: 按出生年干确定化禄/化权/化科/化忌各落在哪颗星
10. 定大限: 每个宫管十年，男女顺逆行不同

### 自检清单
排盘完成后：
- [ ] 命宫位置与出生月+时辰计算一致
- [ ] 五行局与命宫干支匹配
- [ ] 紫微星在合理位置（不会跑到不存在的宫位）
- [ ] 紫微与天府在寅申线对称
- [ ] 十二宫顺排正确（逆时针命宫→兄弟→夫妻→...→父母）
- [ ] 四化与出生年干匹配
- [ ] 大限顺逆与性别+命宫匹配
- [ ] 身宫与命宫按"命逆身顺"规则一致
- [ ] 天魁天钺与年干口诀匹配（甲戊庚牛羊，乙己鼠猴乡...）
- [ ] 星曜亮度覆盖所有有主星的宫位

**如果自检不过，重新排盘。紫微斗数的盘错一处就全错。**

## 解读框架

### 第一步：看命宫
命宫主星决定核心人格。
- 紫微在命 → 有领导气质，自尊心强
- 天机在命 → 聪明灵活，喜欢思考
- 太阳在命 → 热情付出，光明磊落
- ...

### 第二步：看三方四正
每个宫位的重要星曜不仅仅看本宫，还要看：
- **三方**: 地支三合关系对应的两个宫位。例如命宫在寅，寅午戌三合，午宫（财帛）和戌宫（官禄）即命宫的三方。
- **四正**: 三方加上本宫的对宫（迁移宫），共四个宫位。
- **三方四正**: 本宫 + 对宫 + 两个三合宫，是解盘时重点关注的四个宫位。
  - 例: 命宫的三方四正 = 命宫 + 迁移宫 + 财帛宫 + 官禄宫
- 三方四正的星曜会相互影响

### 第三步：看四化在哪个宫
- 化禄的宫 → 顺利、得财、轻松
- 化权的宫 → 掌控、主导、有权力
- 化科的宫 → 名声、贵人、被认可
- 化忌的宫 → 执念、不顺利、最用心的领域

### 第四步：看大限
- 当前大限走到哪个宫
- 那个宫的主星+副星+四化 → 这十年的主题
- 大限命宫的三方四正

### 第五步：看特殊格局
某些星曜组合形成特殊格局，有固定含义：

| 格局 | 组合 | 含义 |
|------|------|------|
| 紫府同宫 | 紫微+天府同宫 | 大格局，领导力+包容力 |
| 日月同宫 | 太阳+太阴同宫（仅在丑未） | 阴阳调和，性格平衡 |
| 杀破狼 | 七杀+破军+贪狼在命宫三方 | 变动大，起伏多 |
| 机月同梁 | 天机+太阴+天同+天梁在命宫三方 | 适合稳定行业（公职/教育） |
| 府相朝垣 | 天府+天相分别在命宫的两夹或三方 | 稳重有福德 |
| 日月反背 | 太阳在酉戌亥子+太阴在卯辰巳午 | 作息异于常人，性格独特 |
| 阳梁昌禄 | 太阳+天梁+文昌+禄存会合命宫三方 | 考试运极佳，功名显达 |
| 火贪格 | 火星+贪狼同宫 | 暴发暴富，横发横破 |
| 铃贪格 | 铃星+贪狼同宫 | 厚积薄发，暗中积累后爆发 |
| 巨日同宫 | 巨门+太阳同宫 | 口才极佳，以言语立足 |
| 月朗天门 | 太阴在亥宫 | 才华外露，女命极佳 |
| 日照雷门 | 太阳在卯宫 | 朝气蓬勃，早年得志 |
| 雄宿乾元 | 廉贞在寅宫 | 刚毅正直，逆境中崛起 |
| 石中隐玉 | 巨门在子或午 | 表面平实内里才华 |
| 明珠出海 | 太阳在卯+太阴在亥 | 阴阳并美，福泽深厚 |

## 星曜亮度（庙旺利陷）

每颗主星在不同地支宫位有固定的亮度等级：庙 > 旺 > 得 > 利 > 平 > 不 > 陷。
同一颗星在庙和在陷的含义天差地别——解读时必须参照亮度。

## 身宫与来因宫

- **身宫**: 后天发展重点，30-40岁后影响力逐渐超过命宫。由出生月+时辰确定（与命宫对称：命宫逆数时辰，身宫顺数时辰）
- **身主**: 按出生年支确定的主星，辅助定位身宫的发展方向
- **来因宫**: 生年天干所在宫位，是四化解盘的原点——同一组四化在不同来因宫含义完全不同

## 对感情的解读

当用户问感情/婚姻时重点看：
1. **夫妻宫主星** → 配偶类型和婚姻状况
2. **夫妻宫四化** → 婚姻的顺利/波折
3. **命宫主星** → 你自身对感情的态度
4. **子女宫** → 子女缘分（也反映亲密关系）
